Access door license

Access door license يرتبط بقرار فتح الباب: reader أو credential يدخل الطلب، controller يطبق rules، ثم lock يعمل مع REX وdoor contact وalarm monitoring.

Access door license is part of the door decision chain: a reader or credential requests access, the controller applies rules, then the lock, REX, door contact, and alarms report the result.

Start Quick Check
01

Quick Definition

لا تقرأ Access door license كقطعة منفصلة. الباب نظام صغير فيه safety, security, power, wiring, software permissions, and fail-safe/fail-secure behavior.

Do not treat Access door license as an isolated part. A door is a small system combining safety, security, power, wiring, permissions, and fail-safe or fail-secure behavior.

02

Why It Matters

خطأ في lock type أو fire interface أو REX قد يمنع الخروج الآمن أو يفتح بابا بدون صلاحية.

A wrong lock type, fire interface, or REX setup can block safe egress or unlock a door without authorization.

03

Where It Appears

يظهر في door schedule، riser، controller I/O list، cable schedule، access levels، fire alarm interface، وSAT door test.

You meet it in door schedules, risers, controller I/O lists, cable schedules, access levels, fire-alarm interfaces, and SAT door tests.

  • No child topics are listed below this row.
04

Practical Examples

  • Check fail-safe/fail-secure behavior against life-safety requirements.
  • Match reader protocol such as OSDP or Wiegand with the controller.
  • Confirm lock power, door contact, REX, emergency release, and fire-alarm release wiring.
  • During testing, prove valid access, denied access, forced door, door held open, and fire release.
DESIGN

Design Considerations

في التصميم، وثق door mode، hardware type، cable cores، power, battery backup، access levels، fire release، وegress path.

In design, document door mode, hardware type, cable cores, power, battery backup, access levels, fire release, and egress path.

SITE

Installation Considerations

في الموقع، راجع alignment، lock voltage، polarity، reader wiring، enclosure tamper، labels، وfire/egress interfaces.

On site, check alignment, lock voltage, polarity, reader wiring, enclosure tamper, labels, and fire/egress interfaces.

TEST

Testing And Commissioning

اختبر valid/invalid credential، REX، forced/held open، fire release، battery backup، event logging، وanti-passback إن وجد.

Test valid and invalid credentials, REX, forced/held-open alarms, fire release, battery backup, event logging, and anti-passback when used.

MISTAKES

Common Mistakes

  • Selecting lock behavior without coordinating life safety.
  • Testing only card unlock and missing forced-door or fire-release behavior.
  • Mixing OSDP/Wiegand wiring or shielding requirements.
TROUBLESHOOT

Troubleshooting Scenario

لو الباب لا يفتح أو يفتح عشوائيا، افحص credential، controller output، lock power، REX، contact state، fire input، وsoftware schedule.

If a door will not unlock or unlocks unexpectedly, check credential, controller output, lock power, REX, contact state, fire input, and software schedule.
