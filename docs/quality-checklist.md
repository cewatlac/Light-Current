# Quality Checklist

- All non-empty CSV rows have pages.
- No duplicate slugs.
- No missing data index entries.
- Homepage, tree, search, and glossary load from static JSON.
- Logo links to `https://anzmatech.com/`.
- Footer credits appear on every page.
- Banned visible phrases are absent.
- Links resolve.
- Quizzes work.
- Theme and language toggles work.
- Pages are responsive and keyboard accessible.
